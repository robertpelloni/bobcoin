# Bobcoin Dashboard

## Crate Status

| Crate | Version | Location | Description |
|-------|---------|----------|-------------|
| `bobcoin-core` | 0.1.0 | `./bobcoin-core` | Core data structures (Block, Transaction) |
| `bobcoin-consensus` | 0.1.0 | `./bobcoin-consensus` | Rhythm consensus logic |
| `bobcoin-privacy` | 0.1.0 | `./bobcoin-privacy` | Privacy primitives (RingCT, Stealth Addresses) |
| `bobcoin-dance` | 0.1.0 | `./bobcoin-dance` | Proof of Dance validation logic |
| `bobcoin-node` | 0.1.0 | `./bobcoin-node` | Main node application/CLI |

## Directory Structure

*   `bobcoin-core/`: Core library.
*   `bobcoin-consensus/`: Consensus library.
*   `bobcoin-privacy/`: Privacy library.
*   `bobcoin-dance/`: Dance validation library.
*   `bobcoin-node/`: Node binary.
*   `Cargo.toml`: Workspace configuration.
*   `VERSION.md`: Global version file.
